# 🚀 Data Redundancy Removal System

This project removes duplicate data entries using Python and Pandas.

## 📌 Features
- Detect duplicate entries
- Remove duplicates based on Email
- Save cleaned data

## 🛠 Tech Stack
- Python
- Pandas

## 📊 How it Works
1. Reads data from CSV file
2. Identifies duplicate records
3. Removes duplicates based on Email
4. Saves cleaned data into a new CSV file

## 🎯 Use Case
Helps maintain clean and accurate data in cloud databases.

## 📁 Files
- app.py → Main Python script
- data.csv → Input dataset
- cleaned_data.csv → Output dataset
